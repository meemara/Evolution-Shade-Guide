INSTALLATION STEPS
-------------------
1. Navigate to the project directory:
   cd /sessions/youthful-focused-hypatia/mnt/Claude-Work/evolution-shade-guide

2. Install new dependencies:
   npm install

3. Configure environment variables for email (create .env file):
   SMTP_HOST=smtp.gmail.com (or your email provider)
   SMTP_PORT=587
   SMTP_USER=your-email@gmail.com
   SMTP_PASS=your-app-password
   SMTP_FROM=Evolution AV <noreply@evolutionava.com>

4. Build the project:
   npm run build

5. Start the server:
   npm run start

USAGE FLOW
----------
1. Client views Shade Guide educational content
2. Clicks on fabric swatch to see details
3. Clicks "Add to Room" button in modal
4. Selects existing room or creates new one
5. Repeats for all desired fabrics
6. Selection bar at bottom shows progress
7. Clicks "Review & Send" to open summary
8. Edits client/project name if needed
9. Selects team members to receive selections
10. Either downloads PDF or sends email
11. Selection data is retained until page refresh or "Clear All"

NOTES FOR DEPLOYMENT
--------------------
- Email delivery requires valid SMTP credentials
- Gmail users should use app-specific passwords
- PDF generation happens server-side with pdfkit
- All selection data is stored in React state (not persisted to database)
- Session data is lost on page refresh
- Consider adding localStorage persistence for future enhancement
